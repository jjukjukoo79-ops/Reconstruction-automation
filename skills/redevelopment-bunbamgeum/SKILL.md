# 정비사업 분담금 계산 스킬

재개발·재건축 관리처분 단계의 핵심 수치를 정확히 계산한다. 절대로 숫자를 임의로
가정하지 않으며, 입력값이 빠지면 계산을 멈추고 해당 값을 요청한다.

## 입력값 (없으면 사용자에게 요청)

**사업 전체값 (비례율 산출용)**
- `분양예정_총수입`: 조합분양 + 일반분양 등 총 분양수입(원)
- `총사업비`: 공사비 + 보상비 + 금융·기타 사업비 합계(원)
- `종전자산_평가총액`: 전체 조합원 종전자산 감정평가액 합계(원)

**조합원 개인값 (개인 분담금 산출용)**
- `개인_종전자산`: 해당 조합원의 종전자산 감정평가액(원)
- `개인_분양가`: 해당 조합원이 신청한 분양주택의 조합원 분양가(원)

## 계산 공식 (도정법 제74조 관리처분계획 기준)

1. **비례율(%)**
   ```
   비례율 = (분양예정_총수입 − 총사업비) ÷ 종전자산_평가총액 × 100
   ```

2. **권리가액(원)**
   ```
   권리가액 = 개인_종전자산 × (비례율 ÷ 100)
   ```

3. **분담금(원)**
   ```
   분담금 = 개인_분양가 − 권리가액
   ```
   - 결과가 **양수 → 추가 납부(분담금)**
   - 결과가 **음수 → 환급금** (절댓값으로 환급액 표기)

## 계산·표기 규칙

- 비례율은 **소수점 둘째 자리까지** 표기(예: 102.37%).
- 금액은 **원 단위 반올림**, 천 단위 콤마 표기(예: 1,234,567원).
- 중간 계산에서 비례율은 반올림하지 말고 **원값 그대로** 권리가액에 적용한 뒤,
  최종 표시용으로만 반올림한다(누적 오차 방지).
- 정확한 산술이 필요하면 아래 검산 스크립트를 실행한다(코드 실행이 가능한 환경일 때).

## 검산 예시 (이 숫자로 로직이 맞는지 항상 자가 검증)

입력:
- 분양예정_총수입 = 1,000억 원, 총사업비 = 600억 원, 종전자산_평가총액 = 392.16억 원
- 어느 조합원: 개인_종전자산 = 3억 원, 개인_분양가 = 5억 원

계산:
- 비례율 = (1000억 − 600억) ÷ 392.16억 × 100 = **102.00%** (소수 둘째자리)
- 권리가액 = 3억 × 1.02 = **3억 600만 원**
- 분담금 = 5억 − 3억 600만 = **1억 9,400만 원 (추가 납부)**

## 조합원 비교표 출력 형식

| 조합원 | 종전자산(원) | 비례율 | 권리가액(원) | 분양가(원) | 분담금/환급(원) | 구분 |
|--------|-------------|--------|-------------|-----------|----------------|------|
| 홍길동 | 300,000,000 | 102.00% | 306,000,000 | 500,000,000 | 194,000,000 | 분담금 |

- 비례율은 사업 전체 단일값이므로 모든 행에 동일하게 적용한다.
- 음수 분담금은 "환급"으로 구분 표기하고 금액은 절댓값으로 적는다.

## 민감도(선택)

요청 시 비례율을 ±1%p, ±5%p 변동시켜 개인 분담금이 어떻게 바뀌는지 표로 제시한다.
(비례율이 1%p 오르면 권리가액 = 개인_종전자산 × 0.01 만큼 증가 → 분담금 그만큼 감소)

## 코드 검산 스크립트 (선택, 코드 실행 가능 시)

```python
def bunbamgeum(총수입, 총사업비, 종전총액, 개인종전, 개인분양가):
    비례율 = (총수입 - 총사업비) / 종전총액 * 100
    권리가액 = 개인종전 * (비례율 / 100)
    분담금 = 개인분양가 - 권리가액
    구분 = "분담금(납부)" if 분담금 >= 0 else "환급금"
    return {
        "비례율(%)": round(비례율, 2),
        "권리가액": round(권리가액),
        "분담금": round(분담금),
        "구분": 구분,
    }

# 검산
print(bunbamgeum(100_000_000_000, 60_000_000_000, 39_216_000_000,
                 300_000_000, 500_000_000))
# → 비례율 102.0, 권리가액 306,000,000, 분담금 194,000,000, 분담금(납부)
```
