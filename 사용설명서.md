# 조합설립 동의서 징구 시스템 — 사용설명서

`동의서_징구_시스템.html` 파일을 **더블클릭**하면 브라우저에서 바로 실행됩니다. (설치·서버 불필요)
모든 데이터는 **이 PC의 브라우저(localStorage)에만** 저장됩니다.

## 처음 한 번 — 설정
1. **⚙️ 설정** 탭 → 사업장명, **사업유형(재개발/재건축)**, **전체 토지등소유자 수**, **전체 토지면적(㎡)** 입력 → 저장
2. 이 값이 있어야 **📊 현황** 탭에서 동의율 요건 충족 여부가 자동 판정됩니다.

## 탭별 기능
| 탭 | 하는 일 |
|----|---------|
| 📊 현황 | 도정법 제35조 동의 요건(소유자 수·토지면적) 실시간 충족/미달 판정 |
| 📝 전자동의서 | 조합원 정보 입력 + 전자서명 → 제출 시 **동의완료**로 명부 등록 |
| ✍️ 전자위임장 | 총회 의결권 등 위임 작성 + 전자서명 → 제출 |
| 🖨 서면양식 | 동의서/위임장 빈 양식 또는 선택 조합원 정보를 채워 **인쇄·PDF 저장** |
| 🔍 스캔 자동입력 | 서면 서류 이미지 업로드 → 성명·주민번호·주소·면적·연락처 **자동 인식** → 확인·수정 후 적용 |
| 👥 조합원 명부 | 전체 목록·검색·상태변경(철회/동의)·삭제, **CSV·백업** |
| ⚙️ 설정 | 사업장 설정, 전체 초기화 |

## 동의율 요건 (도정법 제35조)
- **재개발**: 토지등소유자 **3/4 이상** + 토지면적 **1/2 이상** 동의
- **재건축**: 전체 구분소유자 **3/4 이상** + 토지면적 **3/4 이상** 동의 (＋ 각 동별 과반수는 별도 확인)

## CSV 일괄등록
- `조합원명부_양식.csv` 형식(성명,주민등록번호,주소,부동산소재지,면적,연락처)에 맞춰 엑셀로 작성
- **명부 탭 → 📥 CSV 가져오기** → 일괄 등록(상태: 미동의)
- 엑셀 저장 시 인코딩은 **UTF-8(CSV)** 권장 (한글 깨짐 방지)

## 데이터 백업 (중요)
- 데이터는 이 PC 브라우저에만 있습니다. **다른 PC로 옮기거나 백업하려면**:
  - 명부 탭 → **💾 백업(JSON)** 으로 파일 저장
  - 새 환경에서 **♻️ 복원** 으로 불러오기
- 브라우저 캐시/데이터를 지우면 명부가 사라질 수 있으니 **정기적으로 백업**하세요.

## OCR(스캔 자동입력) 주의
- **이미지(JPG·PNG)** 만 지원합니다. PDF는 화면 캡처 후 이미지로 올려주세요.
- 첫 실행 시 한글 인식 데이터를 내려받으므로 **인터넷 연결**이 필요하고 시간이 걸릴 수 있습니다.
- 인쇄된 글자는 인식률이 높고 **손글씨는 낮을 수 있으니 반드시 확인·수정** 후 적용하세요.
