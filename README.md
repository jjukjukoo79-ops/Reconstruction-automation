# 정비사업 실무 자동화 에이전트

대한민국 도시정비사업(재개발·재건축)을 위한 실무 자동화 에이전트입니다.
「도시 및 주거환경정비법」(도정법) 기반으로 단계 진단·서류 초안·계산을 지원합니다.

## 구성

### 1. 프로젝트 지침 (대화형 에이전트)
- **파일**: [`프로젝트_지침_v2.md`](프로젝트_지침_v2.md)
- 정비사업 전문가 페르소나 + 5단계 절차도 + 단계 판별 로직 + 산출물 생성(안내문/시트/슬라이드).
- 사용법: claude.ai에서 **프로젝트 생성 → 지침(custom instructions)에 파일 내용 붙여넣기**.

### 2. 분담금 계산 스킬 (정확 계산)
- **파일**: [`skills/redevelopment-bunbamgeum/SKILL.md`](skills/redevelopment-bunbamgeum/SKILL.md)
- 도정법 제74조(관리처분계획) 기준으로 비례율·권리가액·분담금/환급금을 계산하고 조합원 비교표 생성.
- 입력값이 없으면 임의로 가정하지 않고 요청하며, 모든 결과에 면책 고지를 표기.
- 사용법: Claude Code의 `~/.claude/skills/`에 두거나, claude.ai 프로젝트에 Skill로 업로드.

### 3. 동의서 징구 시스템 (조합구성 2단계, 단독 실행 HTML)
- **파일**: [`동의서_징구_시스템.html`](동의서_징구_시스템.html) · [사용설명서](사용설명서.md) · [명부 CSV 템플릿](조합원명부_양식.csv)
- 더블클릭하면 브라우저에서 바로 실행(설치·서버 불필요), 데이터는 PC localStorage에만 저장.
- 전자동의서·전자위임장(전자서명), 서면 양식 인쇄, **스캔 자동입력(OCR)**.
- **도정법 제35조 동의율 자동판정**(재개발 3/4+면적 1/2, 재건축 3/4+면적 3/4).
- 개인정보 수집동의, 조합원 명부, CSV 일괄등록/내보내기, JSON 백업/복원, 동의 철회관리.

## 계산 공식 (요약)

```
비례율(%) = (분양예정 총수입 − 총사업비) ÷ 종전자산 평가총액 × 100
권리가액   = 개인 종전자산 × (비례율 ÷ 100)
분담금     = 개인 분양가 − 권리가액   (양수=납부, 음수=환급)
```

## 정비사업 5단계

| 단계 | 명칭 | 핵심 실무 |
|------|------|-----------|
| 1 | 기획·구역지정 | 기본계획, 안전진단, 구역지정 |
| 2 | 조합구성 | 추진위, 조합설립인가, 시공사 선정 |
| 3 | 사업시행인가 | 건축심의, 사업시행계획인가 |
| 4 | 관리처분 | 감정평가, 분양신청, 관리처분계획인가 |
| 5 | 착공·완료 | 이주·철거, 일반분양, 착공, 이전고시, 해산 |
