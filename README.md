# 정비사업 실무 에이전트 대시보드

재개발·재건축 정비사업 실무를 단계별로 지원하는 단일 HTML 대시보드입니다. (도정법 기반)

## 주요 기능

- **상황 진단**: 현재 상황을 입력하면 정비사업 단계(1~5단계)를 자동 진단하고 적합한 에이전트를 추천
- **단계별 에이전트(8종)**:
  - 📋 전자동의서·위임장 시스템 (AI OCR 스캔 자동입력 포함)
  - 📱 SMS 일괄발송 관리
  - 📄 설립등기 서류 자동화 (법원 양식)
  - 🧮 분담금 시뮬레이션 (도정법 §74 비례율·권리가액)
  - 📊 총회 발표자료 PPT 뷰어
  - ⬇️ 총회 PPT 실제 파일(PPTX) 생성
  - 🗺 정비사업 절차도 전체
  - 🔍 AI 스캔 자동입력 (OCR)

## 사용법

`index.html`을 브라우저로 열면 바로 사용할 수 있습니다.

## 정비사업 단계

| 단계 | 내용 |
|------|------|
| 1단계 | 기획·구역지정 |
| 2단계 | 조합구성 |
| 3단계 | 사업시행인가 |
| 4단계 | 관리처분 |
| 5단계 | 착공·완료 |
